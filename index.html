Download

Download GameServer version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted GameServer folder into the libraries folder of your Processing sketches. Reference and examples are included in the GameServer folder.

Keywords. Multiplayer, Game, Server

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.
